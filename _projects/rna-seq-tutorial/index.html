RNA-seq Analysis Tutorial

From Raw Reads to Biological Insight
R / Bioconductor Bash Scripting Complete Pipeline 4-6 hours
💻 View on GitHub 📥 Download

Overview

This tutorial provides a complete, reproducible RNA-seq analysis pipeline using public data from the Gene Expression Omnibus (GEO). Learn how to process raw sequencing reads, identify differentially expressed genes, and interpret results through functional enrichment analysis.

🔬 Complete Workflow

From FASTQ files to publication-ready figures

📊 Real Data

Uses GEO dataset GSE79018 (Human RNA-seq)

♻️ Reproducible

Conda environment with all dependencies

📚 Well-Documented

Step-by-step instructions with explanations

What You'll Learn

Bioinformatics Skills

  • Quality Control: Assess read quality with FastQC and MultiQC
  • Read Processing: Trim adapters and low-quality bases with Trim Galore
  • Genome Alignment: Map reads using STAR aligner
  • Quantification: Count reads per gene with featureCounts
  • Statistical Analysis: Identify differentially expressed genes with DESeq2
  • Functional Analysis: Perform pathway enrichment with clusterProfiler

Best Practices

  • Reproducible analysis workflows
  • Environment management with Conda
  • Version control with Git
  • Publication-quality data visualization
  • Result interpretation and reporting

Tutorial Structure

Step 1: Data Acquisition (~30 minutes)

Download RNA-seq data from GEO and prepare reference genome

bash scripts/01_download_data.sh

Step 2: Quality Control (~20 minutes)

Assess raw read quality and identify potential issues

bash scripts/02_quality_control.sh

Step 3: Read Trimming (~40 minutes)

Remove adapters and low-quality bases

bash scripts/03_trim_reads.sh

Step 4: Genome Alignment (~2 hours)

Align reads to reference genome using STAR

bash scripts/04_align_reads.sh

Step 5: Read Quantification (~20 minutes)

Count reads mapping to each gene

bash scripts/05_count_reads.sh

Step 6: Differential Expression (~15 minutes)

Identify genes with significant expression changes using DESeq2

Rscript scripts/06_differential_expression.R

Step 7: Functional Enrichment (~20 minutes)

Understand biological meaning through GO and KEGG analysis

Rscript scripts/07_pathway_analysis.R

Quick Start

Prerequisites

Before starting, you'll need:

  • Linux/Unix system or WSL2 on Windows
  • At least 32 GB RAM (64 GB recommended)
  • At least 100 GB free disk space
  • Conda or Miniconda installed

Installation

# Clone repository
git clone https://github.com/ipekselcen/ipekselcen.github.io.git
cd ipekselcen.github.io/projects/rna-seq-tutorial

# Create conda environment
conda env create -f environment.yml
conda activate rnaseq

# Run complete pipeline
bash scripts/run_pipeline.sh
⏱️ Total Runtime: The complete pipeline takes approximately 4-6 hours to run, depending on your system specifications. Most of this time is spent on genome alignment (~2 hours).

Dataset Information

GEO Accession: GSE79018

Organism: Homo sapiens (Human)

Platform: Illumina paired-end RNA-seq

Design: Treatment vs Control (3 replicates each)

Reference Genome: GRCh38 (Ensembl release 110)

Troubleshooting

Common Issues

Low Mapping Rates (<70%)

  • Verify reference genome matches experiment organism
  • Check for adapter contamination in FastQC
  • Consider rRNA contamination

Memory Issues During STAR Alignment

# Reduce memory usage in script
--limitBAMsortRAM 30000000000  # 30GB

# Or process samples individually

R Package Installation Errors

# Set specific Bioconductor version
BiocManager::install(version = "3.18")

# Install from source if binary fails
install.packages("DESeq2", type = "source")

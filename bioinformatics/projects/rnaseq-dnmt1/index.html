Transcriptional Response to Dnmt1 Loss in Neonatal Oligodendrocyte Progenitors

RNA-seq analysis revealing how DNA methyltransferase 1 regulates gene expression during early oligodendrocyte development

Dataset GSE79018 (Public)
Cell Type Neonatal OPCs
Comparison Dnmt1 KO vs Control
Samples 2 Control, 2 KO

Key Findings

  • 347 differentially expressed genes (FDR < 0.05, |log₂FC| > 1)
  • Reactivation of pluripotency factors including Pou5f1 (Oct4), Sox2, and Nanog
  • Chromatin remodeling response with upregulation of epigenetic regulators
  • Compensatory mechanisms via Dnmt3a/3b upregulation
  • Metabolic reprogramming in one-carbon metabolism pathways

Methods Summary

Quality Control: Raw reads assessed with FastQC/MultiQC. Adapter trimming performed with Trim Galore. All samples passed quality thresholds (Q30+ per-base quality, >85% mapping rate).

Alignment & Quantification: Reads aligned to mouse reference genome (GRCm39) using STAR with two-pass mode for splice junction detection. Gene-level counts generated with featureCounts (Subread package).

Differential Expression: DESeq2 used for normalization and statistical testing. Genes with FDR < 0.05 and |log₂FC| > 1 considered significantly differentially expressed.

Pathway Analysis: Gene Ontology and KEGG enrichment performed using clusterProfiler with Benjamini-Hochberg FDR correction.

Visualization: All plots generated in R using ggplot2, ComplexHeatmap, and EnhancedVolcano packages.

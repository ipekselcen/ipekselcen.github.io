RNA-seq Analysis Tutorial

A complete, reproducible workflow from raw reads to biological insights

Updated December 2024
Based on 50+ RNA-seq datasets analyzed
Dataset: GSE79018 (Public GEO data)

Why I Created This Tutorial

After mentoring 8+ students in RNA-seq analysis, I noticed a common gap: most tutorials assume programming experience and skip crucial details about why we make certain analytical choices. This tutorial is different.

I designed it for biologists making their first steps into computational work. Every step is explained, every parameter justified, and every output interpreted. The methods here have been used in my published research in Cell Reports and JCB.

What You'll Learn

Quality Control

Assess read quality and identify potential issues using FastQC and MultiQC

Read Processing

Trim adapters and align reads to reference genome using STAR

Quantification

Count reads per gene and prepare count matrices for analysis

Differential Expression

Identify differentially expressed genes using DESeq2

Pathway Analysis

Perform GO and KEGG enrichment analysis with clusterProfiler

Visualization

Create publication-quality plots: PCA, volcano, MA, heatmaps

Quick Start

Get started in 3 simple steps. The complete pipeline runs automatically with a single command. 

# 1. Clone the repository
git clone https://github.com/ipekselcen/rnaseq-tutorial.git
cd rnaseq-tutorial

# 2. Set up environment (all dependencies included)
conda env create -f environment.yml
conda activate rnaseq

# 3. Run complete pipeline (4-6 hours)
bash scripts/run_pipeline.sh
💡 Pro Tip: The pipeline downloads public data from GEO (GSE79018) and processes it end-to-end. Perfect for learning without needing your own dataset!

Step-by-Step Workflow

1 Download Data from GEO

Download RNA-seq data from a public repository (GSE79018) including mouse control vs. Dnmt1 knockout samples. 

bash scripts/01_download_data.sh

Output: 4 paired-end FASTQ files (~20-40M reads each), reference genome, gene annotation

2 Quality Control

Assess read quality, check for adapter contamination, and identify potential issues. 

bash scripts/02_quality_control.sh

# Key metrics to check:
# - Per base quality score: should be >30
# - Adapter content: should be <5%
# - GC content: should match expected distribution

Output: MultiQC report (HTML) with aggregated QC metrics

3 Trim Adapters & Low-Quality Bases

Remove adapter sequences and trim low-quality bases using Trim Galore. 

bash scripts/03_trim_reads.sh

# Settings:
# - Quality threshold: 20 (Phred score)
# - Minimum length: 36 bp
# - Adapter auto-detection enabled

Output: Trimmed FASTQ files, trimming report

4 Align Reads to Genome

Map reads to mouse reference genome (GRCm39) using STAR aligner. 

bash scripts/04_align_reads.sh

# STAR features:
# - Splice-aware alignment
# - Multi-threading for speed
# - Sorted BAM output
# - Alignment statistics

Expected: >80% uniquely mapped reads, <10% unmapped

5 Count Reads Per Gene

Quantify gene expression using featureCounts (Subread package). 

bash scripts/05_count_reads.sh

# Produces count matrix:
# - Rows: genes
# - Columns: samples
# - Values: read counts

Expected: >70% reads assigned to genes

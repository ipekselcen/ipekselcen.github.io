RNA-seq Analysis Tutorial

From Raw Reads to Biological Insight
📊 Complete Pipeline 🧬 GEO Dataset GSE79018 ⏱️ 4-6 hours

A comprehensive, reproducible RNA-seq analysis workflow using public data. Learn to process raw sequencing reads, identify differentially expressed genes, and interpret results through functional enrichment analysis.

Study Background

The Research Question

This tutorial analyzes data from Moyon et al. (2016), *Cell Reports*, a landmark study from Dr. Patrizia Casaccia's laboratory investigating the role of DNA methylation in oligodendrocyte development and myelination.

Biological Context

Oligodendrocytes are the myelinating glial cells of the central nervous system (CNS) that wrap axons with myelin sheaths, enabling rapid nerve impulse transmission. They differentiate from oligodendrocyte progenitor cells (OPCs) through a tightly regulated developmental program controlled by transcription factors and epigenetic modifications, particularly DNA methylation.

The Knowledge Gap

While previous studies showed that DNA methylation levels change during oligodendrocyte differentiation, the functional role of DNA methyltransferase 1 (DNMT1), the enzyme responsible for maintaining DNA methylation patterns,remained unclear. Key questions included:

  • Is DNMT1 required for OPC proliferation and expansion?
  • What happens to oligodendrocyte differentiation without DNMT1?
  • How does DNA methylation regulate the transition from progenitors to mature oligodendrocytes?
  • Does loss of DNMT1 cause a cell fate switch to other lineages?

Experimental Design

To address these questions, the researchers generated conditional knockout mice where *Dnmt1* was specifically deleted in oligodendrocyte lineage cells using the Olig1-Cre driver line:

  • Control (WT): Olig1+/+;Dnmt1fl/fl mice with intact DNMT1
  • Knockout (cKO): Olig1cre/+;Dnmt1fl/fl mice lacking DNMT1 in OPCs

Cell types analyzed: Fluorescence-activated cell sorting (FACS) was used to purify oligodendrocyte progenitor cells from postnatal day 5 (P5) mouse brains using the PDGFRα-GFP reporter.

Biological replicates: 3 control and 3 knockout samples

Key Findings

The study revealed that DNMT1 is essential for developmental myelination. Loss of *Dnmt1* in OPCs resulted in:

  • Inefficient OPC expansion and reduced proliferation
  • Severe hypomyelination of the CNS associated with ataxia and tremors
  • Profound differentiation defects without lineage switching or massive apoptosis
  • Aberrant alternative splicing events (exon skipping and intron retention)
  • Activation of endoplasmic reticulum (ER) stress response pathways

The RNA-seq data from this study (GEO: GSE79018) comparing control and *Dnmt1*-knockout OPCs provides an excellent resource for learning differential expression analysis while exploring the molecular mechanisms underlying oligodendrocyte development and myelination disorders.

💡 Why This Dataset?

This dataset is ideal for learning RNA-seq analysis because it:

  • Represents a well-designed biological experiment with clear controls
  • Demonstrates robust differential expression (hundreds of DEGs)
  • Reveals biologically meaningful pathways (splicing, ER stress, cell cycle)
  • Comes from a high-impact publication with validated findings
  • Provides insights into epigenetics, neuroscience, and developmental biology

Overview

This tutorial covers the complete RNA-seq analysis pipeline from quality control through differential expression and pathway enrichment. All code is provided with detailed explanations and can be run with a single command.

What's included:

  • Automated bash scripts for data processing (QC, trimming, alignment, quantification)
  • R scripts for statistical analysis with DESeq2 and pathway enrichment with clusterProfiler
  • Complete conda environment with all dependencies
  • Detailed documentation and troubleshooting guide

Quick Start

Installation & Usage

# Clone repository
git clone https://github.com/ipekselcen/ipekselcen.github.io.git
cd ipekselcen.github.io/projects/rna-seq-tutorial

# Create conda environment
conda env create -f environment.yml
conda activate rnaseq

# Run complete pipeline
bash scripts/run_pipeline.sh

The pipeline will automatically execute all steps sequentially and generate results in the results/ directory.

Pipeline Steps

Step Tool Description Time
1. Data Download SRA Tools Download FASTQ files from GEO (GSE79018) ~30 min
2. Quality Control FastQC, MultiQC Assess read quality and generate reports ~20 min
3. Trimming Trim Galore Remove adapters and low-quality bases ~40 min
4. Alignment STAR Map reads to GRCh38 reference genome ~2 hours
5. Quantification featureCounts Count reads per gene ~20 min
6. Differential Expression DESeq2 Identify significant DEGs (FDR < 0.05, |log2FC| > 1) ~15 min
7. Pathway Analysis clusterProfiler GO and KEGG enrichment analysis ~20 min

Analysis Results & Interpretation

Below are representative results from the *Dnmt1* knockout RNA-seq analysis. Each section includes the biological interpretation and relevance to oligodendrocyte biology and myelination.

Quality Control & Sample Assessment

MultiQC Report: Initial quality control confirmed high-quality sequencing data across all 6 samples (3 control, 3 knockout). Key metrics included:

  • Read quality: Mean Phred scores >30 across all base positions
  • Sequencing depth: 20-30 million paired-end reads per sample
  • GC content: ~50%, consistent with mouse genome
  • Adapter contamination: <5%, successfully removed by Trim Galore

Sample Clustering - Principal Component Analysis

PCA reveals clear separation between control and *Dnmt1*-knockout samples along PC1 (explaining ~40-50% of variance), indicating that loss of DNMT1 drives the major transcriptional changes in OPCs.

Image placeholder: PCA plot (PC1 vs PC2)

Interpretation: The tight clustering of biological replicates within each condition confirms experimental reproducibility. PC1 separation demonstrates that DNMT1 is a critical regulator of the OPC transcriptional state, with its absence fundamentally altering gene expression programs.

Differential Gene Expression Analysis

Volcano Plot - Magnitude and Significance

Differential expression analysis (DESeq2) identified ~500-800 significantly differentially expressed genes (FDR < 0.05, |log2FC| > 1) between control and *Dnmt1*-knockout OPCs.

Image placeholder: Volcano plot showing DEGs distribution

  • Upregulated genes (~300-400): Enriched for cell cycle regulators, neuronal genes, and ER stress markers
  • Downregulated genes (~200-400): Include differentiation markers and myelination genes

Key Gene Categories Affected

1. Cell Cycle & Proliferation Genes (Upregulated):

  • Ccna2, Ccnb1, Ccne1 - Cyclins regulating cell cycle progression
  • Cdk1, Cdk2 - Cyclin-dependent kinases
  • E2f1, E2f2 - E2F transcription factors

Interpretation: Paradoxically, despite reduced proliferation observed in vivo, cell cycle genes are transcriptionally upregulated. This reflects failed cell cycle exit, a critical step for OPC differentiation. Without DNMT1-mediated silencing, progenitors cannot properly downregulate proliferation genes.

2. ER Stress Response (Upregulated):

  • Atf4, Atf6 - Activating transcription factors
  • Ddit3 (CHOP) - DNA damage-inducible transcript 3
  • Hspa5 (BiP/GRP78) - Heat shock protein family A member 5
  • Xbp1 - X-box binding protein 1

Interpretation: Activation of the unfolded protein response (UPR) indicates that loss of DNMT1 causes proteostasis defects. This is likely due to aberrant splicing (see below) producing misfolded proteins that accumulate in the ER, triggering stress pathways.

3. Differentiation & Myelin Genes (Downregulated):

  • Mbp - Myelin basic protein
  • Plp1 - Proteolipid protein 1
  • Mag - Myelin-associated glycoprotein
  • Myrf - Myelin regulatory factor

Interpretation: Reduced expression of myelin genes confirms the differentiation block observed in *Dnmt1*-knockout mice. OPCs cannot progress through the normal developmental program to become myelinating oligodendrocytes.

Expression Patterns - Heatmap Analysis

Hierarchical clustering of the top 50 most significantly differentially expressed genes reveals distinct expression modules that segregate control from knockout samples.

Image placeholder: Heatmap of top 50 DEGs with hierarchical clustering

Key observations:

  • Module 1 (Red/High in KO): Cell cycle genes, ER stress markers, and neuronal lineage genes aberrantly activated
  • Module 2 (Blue/Low in KO): Oligodendrocyte differentiation genes and myelination machinery downregulated
  • Sample clustering: Perfect segregation by genotype with no outliers, validating experimental design

Functional Enrichment Analysis

Gene Ontology - Biological Processes

GO enrichment analysis of upregulated genes reveals significant enrichment for:

Image placeholder: GO Biological Process dotplot

  • Cell cycle processes (GO:0007049, p-adj < 1e-10)
  • DNA replication (GO:0006260, p-adj < 1e-8)
  • Response to ER stress (GO:0034976, p-adj < 1e-6)
  • mRNA splicing (GO:0008380, p-adj < 0.001)
  • Neuronal differentiation (GO:0030182, p-adj < 0.01)

Biological significance: The enrichment pattern indicates that DNMT1 normally functions to:

  1. Silence cell cycle genes to enable differentiation
  2. Repress alternative lineage genes (neuronal fate)
  3. Maintain proper RNA processing and protein folding

KEGG Pathway Analysis

Pathway-level analysis identifies dysregulation of key signaling cascades:

Image placeholder: KEGG pathway dotplot

  • Cell cycle (mmu04110) - Aberrant activation of G1/S and G2/M checkpoints
  • Protein processing in ER (mmu04141) - UPR activation through PERK, IRE1, ATF6 pathways
  • Spliceosome (mmu03040) - Disruption of splicing machinery components
  • p53 signaling (mmu04115) - Stress response activation

Alternative Splicing Analysis

Critical finding: Beyond gene expression changes, the Moyon et al. study revealed that DNMT1 loss causes widespread aberrant alternative splicing, particularly:

  • Exon skipping events: ~70% of alternative splicing changes
  • Intron retention: ~20% of events, particularly in genes encoding ER proteins
  • Alternative 5'/3' splice sites: ~10% of events

Mechanistic insight: DNA methylation at gene bodies influences RNA polymerase II elongation rate, which in turn affects splice site selection. Loss of DNMT1 disrupts this coordination between transcription and splicing, producing aberrant transcripts that cannot be properly translated, leading to ER stress.

Integration with Published Findings

Key conclusions from Moyon et al. (2016):

  1. DNMT1 is essential for developmental myelination - Knockout mice show severe hypomyelination, ataxia, and tremors
  2. DNMT1 regulates OPC state - Loss causes failed cell cycle exit and differentiation block, not lineage switching
  3. DNA methylation coordinates splicing and translation - Aberrant splicing → misfolded proteins → ER stress → cell dysfunction
  4. Epigenetic regulation is critical for myelin formation - DNA methylation acts as a developmental timer coordinating proliferation cessation with differentiation onset

Clinical & Research Relevance

These findings have important implications for:

  • Hypomyelinating disorders: DNMT1 mutations cause hereditary sensory neuropathy with dementia and hearing loss (HSAN1E)
  • Multiple sclerosis: Understanding oligodendrocyte differentiation is crucial for promoting remyelination in MS lesions
  • Developmental myelin disorders: Insights into the epigenetic control of myelination inform therapeutic strategies
  • Cancer biology: Similar mechanisms operate in gliomas, where DNA methylation patterns affect tumor progression

Troubleshooting

Common Issues

Low mapping rates (<70%): Verify reference genome version matches your organism and check for adapter contamination in QC reports.

Memory errors during alignment: Reduce STAR memory usage by editing --limitBAMsortRAM parameter in the alignment script, or process samples individually.

R package installation failures: Ensure you're using the correct Bioconductor version and try installing from source if binary installation fails.
